# ♟️ N-Queens Solver (Java)

Este projeto implementa a solução para o clássico problema das **N-Rainhas**, utilizando duas abordagens algorítmicas:

- 🔁 **Backtracking**
- 🌳 **Branch and Bound**

O sistema permite executar e comparar os algoritmos com base em métricas de desempenho.

---

## 📌 Objetivo

Posicionar **N rainhas em um tabuleiro NxN** de forma que nenhuma rainha ataque outra.

---

## ⚙️ Pré-requisitos

Antes de executar o projeto, certifique-se de ter instalado:

- ☕ **Java JDK 8 ou superior**
- 💻 **IDE Java** (VS Code, IntelliJ, Eclipse, etc.)

---

## ▶️ Como executar

Siga os passos abaixo para compilar e executar o projeto:

### 🔹 1. Clonar o repositório

```bash
git clone https://github.com/KayllaneAs/TAAL---N-Queens.git
```

### 🔹 2. Executar o programa

Abra a pasta view e execute o arquivo NQueens.java


### 🔹 3. Interagir com o sistema

Após iniciar, o programa solicitará:

```bash
Para iniciar digite um valor N ( ou 0 para sair): 
```

Em seguida, escolha o algoritmo:

```bash
Escolha o algoritmo:
1 - Estratégia Gulosa (Greedy)
2 - Programação Dinâmica (Dynamic Programming)
3 - Voltando Atrás (Backtracking)
4 - Ramificar e Limitar (Branch and Bound)
5 - Comparar TODOS
Opção:
```
